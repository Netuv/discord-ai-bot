## Workflow Pembuatan Artikel

### Step 1: Cari Topik Terupdate + Review Multi-Sumber

**Action:** `researchArticle()` → WebScout engine v4.0
**Tools:** DuckDuckGo + Wikipedia + **Review scraper multi-source**

**Cari Berita:**
- `"[topic] anime news 2026"`
- `"[topic] release date update"`
- `"[topic] announcement trailer"`

**Cari Review & Opini (WAJIB! Jangan cuma MAL!):**
- `[judul] review opinion`
- `[judul] community reaction`
- `[judul] reddit discussion`
- `[judul] anime review site`
- `[judul] rating review`

> ⚠️ **WAJIB:** Ambil dari MULTI-SUMBER: Reddit, AnimeNewsNetwork, forum, Twitter/X, review site.
> Jangan cuma MyAnimeList score! Rangkum KONTEKS opini publik.

**Filter topik:**
- Hype tinggi (sekuel ditunggu, anime populer)
- Rilis dalam 1-2 minggu ke depan
- Breaking news (pengumuman resmi)
- Skip berita lama dan konten NSFW

---

### Step 2: Buat Konten — NARASI (bukan poin-poin!)

#### 2.1 Header yang Menarik

**Rumus headline:** `[Emoji reaksi] + [Judul] + [Info penting] + [Tanda seru]`

**Contoh:**
- `🔥 Summer 2026: Crunchyroll Umumkan Daftar Lengkap!`
- `💥 Re:ZERO Season 4 Part 2: Tanggal Rilis Terungkap!`
- `🎮 GTA 6 Rilis Bulan Depan? Bocoran Terbaru!`
- `😱 Plot Twist! Manga Ini Bakal Tamat Minggu Depan`

#### 2.2 Struktur Artikel — JSON

```json
{
  "title": "[Emoji] Headline max 100 karakter — bikin penasaran!",
  "intro": "Hook 2-3 kalimat. Gaya santai 'aku-kamu', bikin pembaca pengen baca lebih lanjut.",
  "sections": [
    {
      "heading": "🔍 Sub-judul informatif",
      "body": "Paragraf NARASI 4-6 kalimat. Mengalir, santai, analisis + opini multi-sumber. BUKAN poin-poin!",
      "image_query": "Kata kunci gambar (nama anime/judul spesifik)",
      "video_query": "Kata kunci YouTube/trailer/PV"
    }
  ],
  "category": "anime | manga | game | breaking | announcement | general"
}
```

#### 2.3 Gaya Penulisan — WAJIB!

| Aspek | ❌ Hindari | ✅ Gunakan |
|-------|-----------|-----------|
| Format | Bullet/list/poin | **Paragraf narasi** mengalir |
| Tone | Kaku, formal, robotik | **Santai, ngobrol** di Discord |
| Paragraf | > 3 kalimat (tembok teks) | **2-3 kalimat** pendek |
| Hook | "Halo, pada artikel ini..." | **Pertanyaan relatable** / pernyataan berani |
| Transisi | "Selain itu", "Di sisi lain" | **"Nah...", "Bayangin deh...", "Tapi tunggu..."** |
| Sudut pandang | Pasif, impersonal | **"Aku-Kamu"** natural |
| Sumber | Cuma MAL/score | **Multi-sumber: Reddit, ANN, forum, Twitter/X** |
| Review | "Skor MAL 8.5" doang | **Rangkum opini: "Di Reddit bilang...", "Forum X..."** |
| Kata klise | "Kesimpulannya", "Dapat disimpulkan" | **"Intinya...", "Singkatnya...", "Pokoknya..."** |
| Closing | Kesimpulan formal | **Berakhir natural** — tanpa penutup |

**Contoh paragraf NARASI dengan multi-sumber:**
```
Kamu tau gak sih, season ini bakal jadi salah satu yang paling gila
dalam sejarah anime! Di Reddit, banyak yang bilang ini season terpadat
dalam 5 tahun terakhir. Tapi ada juga yang khawatir kualitasnya turun.

Menurut review dari AnimeNewsNetwork, visualnya 'mind-blowing' dan jadi
patokan baru. Tapi di forum MyAnimeLife justru ada yang kritik soal
pacing yang lambat di episode awal. Jadi ya, tergantung selera juga sih.

Yang menarik, konsensus di Twitter/X pecah jadi dua kubu — praise
soundtrack vs kritik character design. Tapi one thing we can all agree,
musim ini WAJIB ditonton!
```

### Step 3: FORMAT OUTPUT KE DISCORD (WAJIB!)

#### 3.1 HEADLINE → Embed

```
[EMBED — warna kategori, TANPA FOOTER/WATERMARK!]
🔥 Summer 2026 Bakal Season Tergila!
Deskripsi: Paragraf intro hook...

ㅤ  ← Invisible spacer (break line visual setelah embed)
```

**Aturan:**
- Embed hanya `title` + `description` + `color`
- **TIDAK ADA** timestamp, footer, atau watermark apapun
- Setelah embed → kirim `ㅤ` sebagai invisible spacer

#### 3.2 Per Section → Message Terpisah

```
**🔍 Deretan Sekuel yang Paling Ditunggu**   ← MESSAGE 1: JUDUL (bold)

Paragraf body narasi di sini...                ← MESSAGE 2: BODY (terpisah!)

🎬 [Judul Video]: https://youtu.be/xxxxx       ← MESSAGE 3: VIDEO (kalau ada)
📸 [attachment gambar]                          ← MESSAGE 4: GAMBAR (kalau ada)

---                                            ← SEPARATOR (kecuali section terakhir)
```

**Aturan WAJIB:**
1. **Judul** = bold `**text**`, dikirim sebagai **MESSAGE TERPISAH**
2. **Body** = narasi, dikirim sebagai **MESSAGE TERPISAH** setelah judul
3. **Video** = link YouTube, dikirim setelah body
4. **Gambar** = attachment, dikirim setelah video
5. **Separator** = `---` antar section (kecuali terakhir)
6. **TIDAK ADA** closing/kesimpulan — artikel berakhir natural
